Some interesting iOS frameworks

Let me know if u found any other really interesting ones. google doc here


realm-cocoa

Realm is a mobile database: a replacement for Core Data & SQLite

SDWebImage

Asynchronous image downloader with cache support with an UIImageView category

BWWalkthrough

BWWalkthrough is a class to build custom walkthroughs for your iOS App

JVFloatLabeledTextField

UITextField subclass with floating labels - inspired by Matt D. Smith's design

MotionBlur

MotionBlur allows you to add motion blur effect to iOS animations.
